# Branding Assets

This folder contains original project branding assets for Codex Discord Rich Presence (`codex-discord-presence`).

- `project-mark.svg`: repository/project icon.
- `social-card.svg`: true-black hero banner with one-line centered title.
- `surface-map.svg`: true-black runtime map with deterministic routing overview.
- `credits-ribbon.svg`: minimal true-black credits banner with embedded app icon for reliable Markdown preview rendering.
- `codex-app.png`: Codex App icon source used for Discord assets and executable branding.
