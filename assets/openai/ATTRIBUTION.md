# OpenAI Brand Assets

This project is designed to work with official OpenAI/Codex branding in your Discord app assets.

## Source

- OpenAI brand page: https://openai.com/brand/

## Usage Notes

- Follow OpenAI brand and trademark policies.
- Upload the official assets to your own Discord application in Discord Developer Portal.
- Use matching keys in `~/.codex/discord-presence-config.json` under `display`.
- For terminal 1:1 logo rendering, set `display.terminal_logo_path` to an official OpenAI logo image file path.
